# ultimate_frisbee_data_analysis

## Overview

This project analyzes player-level statistics from the 2024 College Ultimate Championship to explore how different types of players contribute to on-field impact.

## Dataset
Source: Kaggle (n=1665)
https://www.kaggle.com/datasets/mexwell/2024-college-ultimate-championship-statistics

## Analysis Summary
Exploratory Data Analysis
- Examines distributions and correlations among offensive, defensive, and turnover metrics
- Compares player performance across gender, level, and division
- Shows that player impact is multi-dimensional and not driven by a single statistic
